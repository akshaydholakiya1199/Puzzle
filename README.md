#  15 Puzzle Game

A modern and responsive implementation of the classic **15 Puzzle Game** built using **HTML, CSS, and Vanilla JavaScript**.

The objective is to arrange the numbered tiles from **1 to 15** in the correct order by sliding them into the empty space. The game includes a clean user interface, smooth interactions, and works seamlessly on both desktop and mobile devices.

## Features

*  Classic 4×4 sliding puzzle
*  Random solvable puzzle generation
*  Built-in timer
*  Move counter
*  Keyboard controls (Arrow Keys)
*  Touch and swipe support for mobile devices
*  Win detection with success message
*  New Game button
*  Solve Demo option
*  Modern responsive UI with CSS

##  Technologies Used

* HTML5
* CSS3
* JavaScript

##  How to Run

1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. Start solving the puzzle!

##  Controls

* Click or tap a tile adjacent to the empty space.
* Use **Arrow Keys** to move the blank tile.
* Press **R** to start a new game.
